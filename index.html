Safari extension + macOS companion app

Generate timestamps and summaries for YouTube videos.

Video Timestamps and Summaries for YouTube helps users create chronological timestamps and short summaries directly from supported YouTube video pages.

What it does

The Safari extension adds a right-side sidebar on supported YouTube video pages. The macOS companion app handles Google sign-in and shares that signed-in state with the extension.

How it works

  • The user signs in with Google in the macOS companion app.
  • The extension reads the current YouTube video URL.
  • The app sends the video URL and selected prompt to Gemini.
  • Gemini returns text that the extension displays as timestamps or a summary.

Google authorization

Google authorization is used only so the signed-in user can call Gemini on that user's behalf. The app is not designed to access Gmail, Drive, Calendar, Contacts, or other personal Google account content.
