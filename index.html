Safari extension + macOS companion app

Generate timestamps and summaries for YouTube videos.

Timestamps & Summaries for YT helps users create chronological timestamps and short summaries directly from supported YouTube video pages.

What it does

The Safari extension adds a right-side sidebar on supported YouTube video pages. It reads the available YouTube transcript, creates summaries with Apple Intelligence on the Mac or the user's selected signed-in ChatGPT model, and creates timestamps when ChatGPT is connected.

How it works

  • The extension detects a supported YouTube watch or live page.
  • The extension fetches the available YouTube transcript.
  • Apple Intelligence can create summaries locally without ChatGPT sign-in.
  • If ChatGPT is connected, the selected ChatGPT model creates timestamp candidates from the transcript.
  • The selected ChatGPT model or Apple Intelligence creates the summary, depending on the user's setting.
  • The extension validates timestamp candidates against real transcript cue times.
  • The extension displays the generated results in the sidebar.

Requirements

The app requires Safari on macOS 26 or later and a YouTube video with captions or an available transcript. ChatGPT sign-in is only required for timestamps and optional GPT summaries. Apple Intelligence is optional for local summary generation on compatible Macs. No API key or developer-operated backend is required.
